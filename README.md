組員：電機12 賴柏叡、電機14 陳冠瑄、資財14 葉均芸、運管14 陳羿茹 <br>
主題：電機系畢業預審
## Main Idea
以電機系為目標受眾開發一個能夠判斷畢業要求以及提供修課建議的系統，主要目的如下：
* 降低準大四生對於畢業修課的焦慮
* 電機系人數眾多，可以降低人事成本
* 讓學生可以對於未來修課更有規劃
